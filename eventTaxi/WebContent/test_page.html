Test Seite

Funktionen dieser Demo: Login, Regestrieren, Passwort zurücksetzen, Event anlegen, Event bearbeiten, zu einem Event anmelden und Anmeldung bearbeiten, User löschen, so wie Übersichtslisten.
Zur Nutzung der Demo wird eine SQLite Datenbank benötigt. Weitere Informationen entnehmen Sie der beigelegten Anleitung (pdf).

Die Event-, User- und Anmeldungsübersicht fallen nicht in den Umfang dieser Demo und sind daher nur grundlegend enthalten.

Die Event Info Seite ist nicht in dieser Demo enthalten, daher kommt man von der Event Liste direkt zum Anmeldeformular.
